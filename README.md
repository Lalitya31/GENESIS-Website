# Team Genesis Website

## Overview
This project is a production-quality website UI for Team Genesis, a robotics team based in VIT Chennai. The website is built using React 18 and follows a futuristic, clean, and premium design aesthetic.

## Project Structure
The project is organized as follows:

```
team-genesis-website
├── public
│   ├── index.html         # Main HTML file for the application
│   └── favicon.ico        # Favicon for the website
├── src
│   ├── index.js           # Entry point of the React application
│   ├── App.js             # Main application component
│   ├── components          # Directory for React components
│   │   └── .gitkeep       # Placeholder to keep the directory in version control
│   ├── assets              # Directory for images and fonts
│   │   ├── images
│   │   │   └── .gitkeep   # Placeholder to keep the directory in version control
│   │   └── fonts
│   │       └── .gitkeep   # Placeholder to keep the directory in version control
│   ├── styles              # Directory for CSS styles
│   │   ├── global.css      # Global styles for the application
│   │   └── variables.css   # CSS variables for design tokens
│   └── utils               # Directory for utility functions
│       └── .gitkeep       # Placeholder to keep the directory in version control
├── package.json            # npm configuration file
└── README.md               # Documentation for the project
```
